# Sandbox hra

2D Minecraft
- Lze načíst mapu ze souboru při spuštění:
    - `./nikolnik` + `[jmeno souboru]`
    - `./nikolnik` + `init.map` pokud nebyl uvedený jiný soubor

Ovládání:

- `LMB` - úder
- `RMB` - stavění bloku
- `↑` - skok nahoru
- `←` - pohyb doleva
- `→` - pohyb doprava
- `Ctrl` + `S` - ukládání stavu hry do `save.map`
